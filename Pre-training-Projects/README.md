# Library Management System


This project is a file-based Library Management System developed in Java. It manages books, users, and transactions without using a database by storing data in text files. The system supports core library operations such as adding books, issuing and returning books, user authentication, and tracking borrowing records. It also includes logic for calculating late return fines based on the number of delayed days.



## Features

-Book management (add, remove, search, and display inventory)  
-User management with authentication and role handling  
-Book issuing and return functionality  
-Automatic availability status update for books  
-Late fee calculation based on return date  
-File-based data storage using text files  
-Track borrowed books for each user  
-Admin setup for initial system access


## Technology Used
-Programming Language: Java  
-Core Concepts: OOP (Classes, Objects, Encapsulation)  
-File Handling: FileReader, FileWriter, BufferedWriter, Scanner

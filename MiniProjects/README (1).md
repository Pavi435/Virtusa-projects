# The SocialMedia Content Sanitizer

This project is a Content Sanitizer that filters and cleans text data by detecting harmful or banned words. It also extracts links and helps in monitoring user activity for safe content usage


## Key Features
--Detect and replace banned words  
-Extract links from text   
-Track flagged users  
-Generate cleaned content output  
-Simple text processing system
## Technology Used
-Python  
-File Handling   
-String Processing  
-Basic Data Structures (Lists, Dictionaries)
# The SmartPay Utility Biller
This project is a SmartPay system that calculates electricity bills based on meter readings. It generates a digital receipt including total cost and tax for each customer.
## Features
-Calculate bill based on unit consumption  
-Apply tax automatically  
-Generate digital receipt  
-Input validation for meter readings  
-Continuous user input system
## Technology
-Java  
-OOP Concepts (Interface, Classes)   
-Scanner (User Input Handling)
